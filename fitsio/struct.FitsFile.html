Wrapper around a C fitsfile pointer.

This handles opening, creating and automatically closing (through the Drop trait) the file.

All subsequent file access is controled through this object.

fn current_hdu_number(&self) -> usize

Returns the current HDU number, 0-indexed

The FITS standard is 1-indexed (thanks Fortran), so this function converts the indexing to be 0-based so the primary HDU is at index 0.

Examples

let f = FitsFile::open("../testdata/full_example.fits").unwrap();
assert_eq!(f.current_hdu_number(), 0);
